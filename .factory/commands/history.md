# Git History Analysis

Analyze git history for: $ARGUMENTS

## Execution

Spawn `git-history-analyzer` subagent to understand:
- Why code looks the way it does
- Who has context on specific areas
- Patterns in how code has evolved
- Related changes and refactors

## Use Cases
- Before modifying unfamiliar code
- Investigating why code looks "weird"
- Finding who to ask about specific areas
- Understanding past decisions
